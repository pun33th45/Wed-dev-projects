- Custom 404 error page
- Expense tracker
- Password generator

## Tech Stack

No build step is required. Open any project folder and launch its `index.html` file in a browser.

Each project is standalone. Use these examples as practice references for layout, DOM manipulation, styling, and basic frontend interactions.
